# 레퍼런스 갤러리 인덱스

사이트 갤러리(`docs/js/gallery-data.js`)의 실제 프롬프트를 담은 카테고리 파일 라우팅 인덱스입니다. 기본으로 모든 파일을 열지 말고, 아래에서 가장 가까운 카테고리 파일 1개(혼합이면 2~3개)와 `craft.md`만 읽습니다.

각 `gallery-*.md`에는 해당 카테고리의 완성 프롬프트 10개, 이미지 미리보기, 사이즈 메타가 들어 있습니다.
