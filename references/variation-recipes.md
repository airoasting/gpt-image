# 변주 레시피

"시안 여러 개", "차별화되게", "컨셉부터 잡아줘", "무드만 바꿔" 요청에 쓴다. 7블록이 하나의 의도를 정밀하게 옮기는 도구라면, 이 파일은 **한 소재를 여러 갈래로 벌리는** 도구다. 옮겨 적기가 아니라 발상이 필요할 때 연다.

기본기: 막연히 "다르게"가 아니라 **한 번에 한 가지만 바꾼다.** 나머지 블록은 그대로 두고 한 가지 결(사조·대비·팔레트·감정·구도)만 바꿔 여러 장을 내면, 시안들이 서로 뚜렷이 다르면서도 한 세트로 읽힌다. 두세 가지를 동시에 흔들면 세트감이 무너진다.

## 다섯 가지 레시피 (하나만 골라 반복)

| 레시피 | 언제 | 손대는 블록 |
|---|---|---|
| ① 사조 갈아끼우기 | "느낌을 통째로 바꿔줘", 스타일 다변화 | 스타일과 매체 + 색 + 구도 |
| ② 상반 요소 배치 | "밋밋해", 긴장감이 필요할 때 | 맥락 + 빛 |
| ③ 무드→팔레트 변환 | "무드만 바꿔", 색 다변화 | 스타일(HEX) + 빛 |
| ④ 감정→시각 신호 | 추상적 소재를 그림으로 | 주 피사체 + 디테일 |
| ⑤ 같은 소재 다른 구도 | 한 피사체를 여러 컷 | 구도와 비율 |

---

## ① 사조 갈아끼우기

사조 이름만 넣으면 모델이 평균값을 낸다. 사조를 **형태·색·글자·짜임 네 가지로 풀어** 적는다.

| 사조 | 형태 | 색 | 글자 | 짜임 |
|---|---|---|---|---|
| 스위스 인터내셔널 | 격자, 직각, 여백 | 흑·백 + 원색 하나 | 산세리프, 왼쪽 맞춤 | 비대칭 균형, 여백 넉넉 |
| 바우하우스 | 원·삼각·사각 기본 도형 | 빨강·노랑·파랑 | 기하 산세리프 | 도형 조합, 대각 흐름 |
| 아르데코 | 대칭, 계단형, 방사선 | 골드 + 딥 네이비·에메랄드 | 고대비 세리프 | 중앙 대칭, 수직 강조 |
| 멤피스 | 무작위 도형, 지그재그 | 파스텔 + 형광 충돌 | 굵고 장난스러운 | 흩뿌린 배치, 패턴 과잉 |
| 미니멀 | 단순 실루엣, 한 요소 | 무채 + 포인트 하나 | 얇은 산세리프 | 극단 여백, 단일 초점 |
| 우키요에 | 평면, 굵은 윤곽선 | 인디고·주홍·황토 | (없음) | 대각 흐름, 여백 구름 |
| Y2K 팝 | 크롬, 버블, 렌즈플레어 | 홀로그램, 형광 그러데이션 | 늘어난 산세리프 | 중앙 오브젝트, 반사 |

적용: 스타일과 매체 블록에 위 네 가지를 문장으로 옮긴다. 예) "스위스 인터내셔널 편집 디자인. 12열 격자, 왼쪽 맞춤 산세리프, 흑백에 빨강 포인트 하나, 상단 여백 넉넉히."

## ② 상반 요소 배치

한 화면에 반대되는 두 성질을 **자리를 나눠** 담으면 긴장이 생긴다. 섞으면 탁해지니 어느 것을 어디에 둘지 못 박는다.

| 상반 짝 | 한쪽(어디에) | 반대쪽(어디에) |
|---|---|---|
| 따뜻함 ↔ 차가움 | 피사체를 따뜻한 골든 톤 | 배경·그림자를 차가운 블루 |
| 매끈함 ↔ 거침 | 주 피사체를 깔끔한 렌더 | 배경·질감을 거친 종이·붓결 |
| 고요 ↔ 긴장 | 넓은 여백으로 정적 | 한 점에 강한 대비·시선 |
| 고급 ↔ 일상 | 재질을 무광 프리미엄 | 소품을 흔한 생활 물건 |
| 질서 ↔ 흐트러짐 | 배경을 정연한 격자 | 전경을 흩어진 파편·움직임 |

적용: 맥락·빛 블록에서 "이건 이쪽, 저건 저쪽"으로 위치를 지정한다.

## ③ 무드→팔레트 변환

"무드만 바꿔"에 강하다. 추상 무드를 **HEX 3~5개와 60/30/10 비율**로 바꿔 적는다.

| 무드 | 60% 바탕 | 30% 보조 | 10% 포인트 |
|---|---|---|---|
| 조용한 럭셔리 | `#EDE7DD` 웜 그레이 | `#2B2B2B` 차콜 | `#B8975A` 브러시드 골드 |
| 새벽 무드 | `#0F1B2D` 딥 네이비 | `#3A5068` 슬레이트 | `#E8A15A` 앰버 |
| 봄 파스텔 | `#FBEFF2` 블러시 | `#CFE3D4` 민트 | `#F2B84B` 옐로 |
| 다크 테크 | `#101418` 잉크 | `#1F6FEB` 일렉트릭 블루 | `#00E0A4` 네온 민트 |
| 레트로 인쇄 | `#F4ECD8` 크림 | `#C0392B` 브릭 레드 | `#2C6E63` 틸 |

적용: 스타일 블록에 HEX를 그대로 넣고, 빛 블록의 색온도를 팔레트에 맞춘다(따뜻하면 3200K, 차가우면 6500K).

## ④ 감정→시각 신호

"설렘", "몰입", "안도" 같은 감정 소재는 형용사로 두면 안 나온다. **눈에 보이는 신호로 바꿔** 적는다.

| 감정 | 눈에 보이는 신호(주 피사체·디테일에) |
|---|---|
| 설렘 | 번지는 밝은 빛, 위로 떠오르는 입자, 밝아지는 화이트 밸런스 |
| 몰입 | 주변 흐림, 한 점만 선명, 조용한 배경 |
| 안도 | 내려간 어깨, 따뜻한 확산광, 열린 여백 |
| 긴장 | 좁은 프레임, 또렷한 그림자 경계, 기울어진 수평선 |
| 그리움 | 필름 입자, 바랜 채도, 창가 역광 |

---

## ⑤ 같은 소재 다른 구도

피사체·스타일은 그대로 두고 구도와 비율만 바꾼다.

- 아주 넓게(환경 강조) → 중간(피사체+맥락) → 얼굴/디테일 가까이
- 정면 → 낮은 3/4 → 위에서
- 세로 3:4(포스터) → 정사각 1:1(SNS) → 가로 16:9(썸네일)

## 여러 장 낼 때

"시안 N개" 요청:

1. 바꾸지 않을 블록을 고정한다(피사체·맥락·정확성 조건).
2. 레시피 하나를 골라 N개 값을 뽑는다.
3. 각 값으로 한 블록만 바꿔 N개 프롬프트를 만든다. **한 장에 한 컷씩** 별도로 내고, 한 캔버스에 여러 컷을 몰아넣지 않는다.
4. 배치로 낼 때는 한 줄에 한 컷씩 jsonl로 낸다(스키마는 SKILL.md 배치·양산 모드).

마지막 점검: N개가 서로 "다른 그림"이면서 "한 세트"로 읽히는가. 한 번에 두 레시피를 섞으면 세트감이 깨진다.
