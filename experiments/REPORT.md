# 실측 검증 리포트 — 사이트 갤러리 프롬프트 정합 (2026-07-05)

카테고리당 1개씩 10개 컷을 뽑아, 각 프롬프트가 실제 이미지를 얼마나 충실히 재현하는지 비전 채점(고정 루브릭 `RUBRIC.md`, 독립 채점자 10인 병렬)했다. 채점 원본은 `scores/gallery.scores.jsonl`, 표본은 `select_sample.py`로 재현한다.

이 리포트는 스킬 규칙(네거티브 정책·검증기 설계)의 근거를 실측으로 확인한다.

## 표본

`ref-6`(웹툰) · `ref-16`(일러스트) · `ref-26`(게임) · `ref-36`(시네마틱) · `ref-46`(포스터) · `ref-56`(제품·브랜드) · `ref-66`(인물) · `ref-76`(건물) · `ref-86`(인포그래픽) · `ref-96`(UI·대시보드).

## 집계

| 지표 | 값 | 비고 |
|---|---:|---|
| goal_fit (프롬프트↔이미지 일치) | **5.00 / 5** | 10/10 만점 |
| text_accuracy (렌더 텍스트) | **4.50 / 5** | 텍스트 있는 6컷, 전부 ≥4 |
| layout (영역·구성) | 5.00 / 5 | |
| material_realism (재질) | 5.00 / 5 | |
| neg_rendered (네거티브 렌더) | **0 / 10** | |
| 합격(qa 평균 ≥4 AND text ≥4) | **10 / 10** | |

## 결론

1. **프롬프트-이미지 정합은 만점 수준이다** (goal_fit 10/10). 앞선 "실제 픽셀 기준 프롬프트 정합" 작업이 실측으로 확인됐다. 표본에서 "없는 요소를 요구"하거나 "핵심 요소 누락" 사례 0건.

2. **한글 텍스트는 4.5/5로 안정적이나 천장은 아니다.** 감점은 전부 경미했다: 포스터 `ref-46`의 "전국" 한 단어 누락, 제품 `ref-56`의 SPF/PA fine print 흐림, 웹툰 `ref-6`의 약병 라벨 소형 잉여 글자. 치명 오탈·유령 텍스트 블록·전면 창작은 0건. → **따옴표 고정 정책은 유효**. 남은 결함은 전부 **소형 밀집 텍스트(fine print, 라벨)**에 몰려 있다.
   - **룰북 반영:** 밀집·소형 텍스트는 캔버스 큰 변(2048) 또는 카피 수 축소로 대응(SKILL.md 사이즈 락, craft.md §1). 치명 카피만 따옴표로 지키고 fine print는 최소화한다.

3. **꼬리 네거티브는 이 표본에서 무해했다** (neg_rendered 0/10, goal_fit 손상 0). 사이트 프롬프트의 "의미 없는 글자... 피합니다" 같은 꼬리 배제 문구가 이미지에 글자로 렌더되지 않았고, 배제 대상이 잘못 렌더되지도 않았다. → **검증기 설계 부합**: 한국어 배제형(`W-NEG-KO`)을 error가 아니라 warning으로 둔 판단이 데이터와 일치한다. 긍정형 재서술은 원칙으로 권장(품질·명료성)하되, 꼬리 배제 문구를 실격 사유로 강제하지는 않는다.

## 한계

- N=10(카테고리당 1). 천장 효과로 카테고리 간 변별력은 제한적이다. 이 리포트는 "정합이 무너지지 않았다"와 "네거티브가 무해했다"를 확인하는 용도이며, 캔버스 크기 같은 주효과의 정량 비교(대조군 크기 실험)는 별도 생성 실험이 필요하다.
- 채점은 단일 표본 1회. 반복·교차 채점은 후속 과제.
